# Barbería Urbana - Landing Page

## Descripción
Proyecto de Landing Page para una barbería moderna. Permite revisar servicios, precios, el equipo de barberos y agendar un turno online.

## Estructura del Proyecto
- `index.html`: Punto de entrada principal.
- `src/css/styles.css`: Estilos y diseño responsive.
- `src/js/main.js`: Validación del formulario de reserva.
- `src/assets/`: Imágenes y recursos visuales.

## Instrucciones de Visualización
Para ver el proyecto, clona este repositorio y abre el archivo `index.html` en cualquier navegador web.
